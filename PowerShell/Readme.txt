Автоматизация обновлений для базы Polomars

Обновлять базу можно отдельно по каждому городу (base_kazan, base_rostov и т.д.),
так и одним скриптом all_city (одновременно по разным городам и нескольков
промежуточных обновление).
Программ скачивает с фтп обновление, разархивирует файлы копирует в нужные папки,
запускает файлы на обраотку программой Datmngr.
