Playbook с названием deploy_apache_web (в виде дерева ролей)
Запускается при помощи playbook_roles.yml

Задача playbook:

1. Установить последнюю версию Apache в зависимости от вида
Linux, RedHat или Debian (для компактности текста программы
группировка кода по блокам)

2. Запустить сервис Apache, причем при перезапуске системы он будет
автомтически запущен.

3. Копирование файла index.j2 с включеннымив текст перемнными и
одновременная его генерация в файл index.htm на сервере
в качестве переменной пути переменная destin_folder (описана
в другом файле). Права владения файлом mode=0555 (первое значение стикер,
остальные значение r-x для юзера, группы и остальных).

4. Если файлы обновились перезапускается Linux, используется handler (в отдельном
файле).

5. Аналогично копируются остальные файлы из списка (в данном случае mytext.txt).

6. Если файлы обновились перезапускается Linux, используется handler (в отдельном 
файле).
