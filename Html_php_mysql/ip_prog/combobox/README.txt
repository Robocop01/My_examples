Поле ввода с выпадающим списком by c-h-i-z-h

компонента прадназначена, для создания одного или нескольких полей ввода с выпадающим списком
где список вариантов берётся из таблицы mySQL (для использования компоненты в ее первозданном виде:
на сервере должна быть поддержка PHP + MySQL, у клиента поддержка CSS + JavaScript)

Протестированно на браузерах Firefox, Opera, Chrome, IE, Safary
  Firefox - идеально
  Opera - идеально
  Chrome - идеально
  IE - в целом работает но не так красиво как задумано
  Safary - отображается правильно, но не работает выбор из списка стрелками клавы(выбор только мышью)

ИСПОЛЬЗОВАНИЕ:

1)  папку combobox содержащую этот файл и файлы combo_boxes.php и combo_box_lib.js кладем,
    в корень папки со страницей содержащей компоненту.
	
2)  в index.php (можно и не index) перед выводом в браузер HTML тега закрывающего "</head>" вписываем
    require_once('combobox/combo_boxes.php');

3)  открываем файл combo_boxes.php и видим там подключение к базе mySQL с которой берутся данные -
    меняем на свой вариант

	далее, видим описание некоторых параметров наших COMBOBOXов
	
    $comboboxes = 2; #количество COMBOBOXов на странице
    $combobox[1]['font-size'] = '14'; #шрифт в поле ввода и списке (остальные параметры шрифта ниже теге <style>)
    $combobox[1]['width'] = '200'; #ширина нашего COMBOBOXа (высота определяется шрифтом)
    $combobox[1]['top'] = '4'; #расстояния от верха страницы до начала поля ввода(нужно для позиционирования списка)
    $combobox[1]['dbtable'] = 'myDBtable1'; #таблица базы данных из которой получать данные для списка
    $combobox[1]['dbfield'] = 'myDBfield1'; #поле таблицы базы данных из которого получать данные

	правим на свой вариант
	
4)  в нужном месте страницы (где должен висеть наш ComboBox) вписываем код:

    echo createcombobox($combonum,$name,$value,$formid);
	где
	$combonum - номер COMBOBOXа
	$name - имя COMBOBOXа (совпадает с именем отправляемой на сервер переменной при SUBMITе)
	$value - преднаписаный текст в поле ввода (пустые ковычки дадут пустое поле ввода)
	$formid - ID формы (нужно только если при выборе значения из списка форма отправлялась на сервер,
	если не нужно то пустые ковычки)
