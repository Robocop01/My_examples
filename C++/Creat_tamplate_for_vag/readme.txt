Программа для формирования шаблонов для сообщений (в файле 139700 должен 
быть проставлен признак 1, если вагон арендованный).

Список арендованных вагонов в файле arenda
Пользователь прописывает вагоны в файле 139700, причем номера вагонов могут быть 
записанны в любой последовательности,
через пробел, запятую, в строчку, столбик, как угодно.
Затем файл 139700 перезаписывается в нужном виде.
