Краткое описание файлов:

MgCaO2.cpp - программа для генерации КФ32 в два цвета без ограничений на заряды, так как они везде +2. Вводить сначала заряд, затем его количество, в избыточных запросах ввести просто 0. genCaMg.exe исполняемый модуль этой программы. На 9.10.2009 удалось сгенерировать КФ соотношений 1:31, 2:30, 3:29, 4:28, 5:27, 6:26, 7:25, 8:24, 9:23,10:22, 11:21, 12:20, 13:19, 14:18 (28,5 Gb). Не рискнул 15:17 и 16:16. Для ускорения расчета создан модернизированный вариант MgCaO2_oct.cpp, в расчете используются уникальные октаэдры в центре. Так же создана еще более оптимизированная программа(в каталоге genCaMg3, там есть файл для уникальной внутренней раскраске 14 позиций) для генерации КФ, число одинаковых вариантов сведено к минимуму, за счет генерации унакального куба с октаэдром (скорость расчета возросла в 13 раз).

В каталоге MaCaO2 - результаты расчетов.


Uslforgsovm.cpp  программа для написания условий для взаимопроникающих КФ32 (список октаэдров). Uslov.txt  результат (код программы, взаимопроникающие октаэдры).

Vzgsuniver.cpp  универсальная программа отсева (отсеиваются взаимопроникающие и геометрически совместимые КФ32). В программе достигнут оптимальный уровень быстродействия. Univz_gz.exe  исполняемый модуль программы, входные данные количество строк (возможно до 1700000 строк). Для работы программы необходим файл 139700 (строки). Соотношение 7:25 (for 2 color) обрабатывало 1ч54мин (процессор amd barton 2600xt).
Далее на процессоре g1820 были отсеяны варианты для 7:25 (00:03), 8:24 (00:27), 9:23 (03:05), 10:22(15:36), 11:21 (60:42)
